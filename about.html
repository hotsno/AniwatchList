AniwatchList

About AniwatchList

AniwatchList gets your AniList "Watching" list and displays each item with a button linking to the next episode on Aniwatch. Note: At the time of writing Aniwatch is closed for registration.

It does not, however, update your AniList for you. I recommend using a great extension called MAL-Sync for this (despite the name it syncs with AniList as well).

This project is open source on GitHub and is my first time making a website, so don't expect much from it. Please star the project if you enjoy using the site!
